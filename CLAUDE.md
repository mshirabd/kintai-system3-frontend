# CLAUDE.md - Claude Code 指示ファイル

## 作業開始時のルール

会話を開始したら、作業前に必ず以下を実行すること：

```bash
git pull origin main
```

リモートの変更をローカルに反映するため。

## デイリーノートへの記録ルール

### ファイルパス
`/Users/m-shirakawa/Library/Mobile Documents/iCloud~md~obsidian/Documents/Obsidian Vault/Journal/YY.MM.DD.md`（例: `Journal/26.04.01.md`）

### 記録タイミング
以下のいずれかで作業内容をデイリーノートに追記すること：
- ユーザーから「記録して」「デイリーノートに追記して」と指示されたとき
- セッション終了時にユーザーから確認されたとき

### 記録フォーマット
```
## [作業タイトル]（HH:MM頃）

- 実施した作業を箇条書きで簡潔に記録
- 作成・編集・削除したファイルがあれば記載
- 発生した問題と解決策があれば記載
- 次回の課題・TODO があれば記載
```

### 注意事項
- デイリーノートが存在しない場合は新規作成してから追記する
- 既存の記録は上書きせず、末尾に追記する
- 機密情報・パスワードは記録しない
